# Feiling

一个以 **桌面陪伴型桌宠** 为目标的桌面应用项目，当前采用 **WPF + 精灵帧动画** 方案实现基础交互与行为系统。

## 项目定位

Feiling 的核心目标不是做一个静态挂件，而是做成一个具有陪伴感、存在感和基本行为逻辑的桌面角色应用。

当前项目重点放在：

- 稳定的桌宠外壳
- 角色待机与移动动画
- 模式切换与行为控制
- 桌面端的自然交互体验

## 当前能力

- 透明无边框桌宠窗口
- 基于精灵帧的待机动画循环
- 三种行为模式：
  - 固定位置
  - 闲逛
  - 追逐鼠标
- 左右移动朝向自动镜像
- 全身拖拽
- 右键菜单与基础交互项

## 技术路线

- 桌宠壳：WPF
- 角色表现：精灵帧动画
- 当前运行模式：桌面本地应用

## 项目亮点

- 不是单纯 UI 练习，而是把 **角色设定、动画素材、运行时逻辑、桌面交互** 组合成一个完整的应用原型
- 有明确的行为模式设计，而不是只展示静态图片
- 适合作为 **桌面应用 / 创意交互 / AI 内容整合型产品原型** 展示
- 可与 AI 绘画、AI 视频、角色设定工作流进一步结合

## 目录说明

- `desktop/FeilingPetShell`
  - WPF 桌宠本体
- `assets/characters/feiling`
  - 角色素材
- `docs`
  - 角色设定、出图与动画相关文档
- `scripts/start_feiling.ps1`
  - 启动脚本
- `scripts/stop_feiling.ps1`
  - 停止脚本

## 运行方式

先确保本机安装 `.NET 8 SDK`。

启动：

```powershell
.\scripts\start_feiling.ps1
```

停止：

```powershell
.\scripts\stop_feiling.ps1
```

也可以直接使用根目录中的批处理脚本快速启动和停止。
